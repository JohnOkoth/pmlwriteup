My analysis for prediciting Classe variable involved extracting the data for Human workout provided by Groupware@less. The data had respective predictor variables for different types of workout. I went through the process of pruning the variables that had a higher amount of missing data, and irrelevant data by creating a function that did the clean up. There are two functions that I created, one for cleaning up the data (preprocess) and the other for cleaning and partitioning the data (partition). The partitioned data from "train" generated both the training and testing data, with the randomForest package used to model the training data under the Caret package. I later used the test data to predict the Classe outcome, and performed the degree of accuracy with reference to the Classe variable from the testing data.This was done under the confusionMatrix function. The test data that I uploaded from the prescribed website, served the purpose of predicting the 20 cases as was assigned. The cross validation generated 25 resamples under the bootstrap method without replacement. The caret function for creating the cross validation, chose the value by default, and under the ressampling process, respective parameters were obtained with the highest value of accuracy estimate chosen under modeling tree number 27. This model was the best since it gave an accuracy of 99.5%, with an output sample error of 0.06796941% in relation to the function I computed in the R-Code under line 66 and 69 respectively in the .Rmd file.Under the confusionMatrix function, there were only three errors predicted i.e two B's and one C giving proof that the model was robust in predicting Classe.
